## 8085 Microprocessor (Basics)

8085 ek **8-bit microprocessor** hai.
Iska kaam simple hai:
- instructions lena
- process karna
- output dena

Key points jo yaad rakhe:
- 8-bit data bus
- 16-bit address bus (64KB memory)
- Simple architecture
- Mostly basic systems aur learning purpose ke liye use hota hai

8085 ko samajhna important hai kyunki
**yahin se microprocessor ka base clear hota hai**.

---

## 8086 Microprocessor (Next level)

8086 ek **16-bit microprocessor** hai,
jo 8085 se kaafi powerful hai.

Important cheezein:
- 16-bit data bus
- 20-bit address bus (1MB memory)
- Do main parts:
  - BIU (Bus Interface Unit)
  - EU (Execution Unit)

Yahin se:
- segmentation
- pipelining
- real mode concepts start hote hain

Exam point of view se:
**8086 = MOST IMPORTANT**.

---

## 8086 Processor (Architecture understanding)

8086 ka architecture thoda complex hai,
but logic simple hai:

- BIU → memory se instruction laata hai
- EU → instruction execute karta hai

Is separation se:
- speed better hoti hai
- overlapping possible hoti hai (fetch + execute)

Yeh concept exam me likhne layak hai.

---

## Intel vs Motorola (Basic difference)

Aaj ye bhi samjha ki
**Intel aur Motorola processors alag philosophy follow karte hain**.

- Intel:
  - Little endian
  - More popular in PCs
- Motorola:
  - Big endian
  - Cleaner instruction set (comparatively)

Exact detail baad me dekhenge,
but exam ke liye difference yaad rakhna enough hai.
